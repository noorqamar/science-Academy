You are a senior UI/UX designer and full-stack web developer. Build a complete, premium, modern, highly professional online Science Tutoring Academy website.

PROJECT GOAL

Create a scalable online tutoring academy rather than a personal tutor portfolio.

The academy initially teaches:

* Physics
* Mathematics
* Biology
* Chemistry

Supported curricula should include:

* O Level
* IGCSE
* GCSE
* A Level
* IB
* AP

The architecture must allow additional subjects, curricula, tutors, courses, and resources to be added later without redesigning the website.

The website should feel like a premium modern EdTech company and established online academy.

It must NOT look like:

* a cheap tuition-center template
* a children's school website
* a generic WordPress education theme
* an individual tutor portfolio
* a crowded marketplace

DESIGN DIRECTION

Create a sophisticated, trustworthy, academically premium visual identity.

Visual style:

* Modern EdTech
* Premium
* Scientific
* Minimal
* Spacious
* Professional
* Conversion-focused
* Clean typography
* Excellent hierarchy
* Sophisticated micro-interactions

Use generous whitespace.

Avoid excessive gradients, huge shadows, cartoonish graphics, unnecessary animations, and excessive use of colors.

COLOR SYSTEM

Primary:
Deep Navy approximately #071A2E

Secondary:
Professional Royal/Electric Blue approximately #2563EB

Background:
White and very light cool blue-gray

Optional accent:
Soft cyan or emerald

Subject identification colors may be used subtly:

Physics = Blue
Mathematics = Purple
Biology = Green
Chemistry = Orange

These colors should only help differentiate subjects. The academy's core brand should remain consistent.

TYPOGRAPHY

Use a premium modern font combination such as:

Headings: Manrope or Plus Jakarta Sans
Body: Inter

Headlines should be large, confident, clean, and easy to read.

TECHNOLOGY

Use a modern production-quality architecture.

Preferred frontend:

* Next.js
* React
* TypeScript
* Tailwind CSS

Use reusable components and a clean project structure.

The website must be fully responsive for:

* Desktop
* Laptop
* Tablet
* Mobile

Create proper hover, active, focus, loading, and interactive states.

Accessibility and good semantic HTML should be considered.
